# Login with Facial Gesture

Login Portal that Verifies user based on facial gesture (eye-blink) recognition
Liveness Detection Algorithm
Effective on face spoofing

# Used Technology 
Tkinter
OpenCV/Haar-Cascades
Tenser Flow / keras
